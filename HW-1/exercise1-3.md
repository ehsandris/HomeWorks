# پاسخ سوالات

## سوال 1: حداقل 5 نمونه از کاربردهای هوش مصنوعی در زندگی روزمره

1. **دستیارهای هوشمند**
   دستیارهایی مانند ChatGPT و Google Assistant با استفاده از هوش مصنوعی به سوالات پاسخ می‌دهند و کارهای مختلف را انجام می‌دهند.

2. **پیشنهاد فیلم و موسیقی**
   سرویس‌هایی مانند Netflix و Spotify با تحلیل سلیقه کاربران، فیلم‌ها و موسیقی‌های مناسب را پیشنهاد می‌کنند.

3. **ترجمه زبان**
   ابزارهایی مانند Google Translate با استفاده از هوش مصنوعی متن و گفتار را بین زبان‌های مختلف ترجمه می‌کنند.

4. **تشخیص چهره**
   بسیاری از گوشی‌های هوشمند از هوش مصنوعی برای تشخیص چهره و باز کردن قفل دستگاه استفاده می‌کنند.

5. **فیلتر کردن هرزنامه (Spam)**
   سرویس‌های ایمیل با استفاده از هوش مصنوعی ایمیل‌های تبلیغاتی یا مخرب را شناسایی کرده و به پوشه Spam منتقل می‌کنند.

## سوال 3: طراحی سیستم تشخیص هرزنامه (Spam)

### 1. چه داده‌هایی جمع‌آوری می‌کنید؟

* متن ایمیل
* عنوان (Subject)
* آدرس فرستنده
* وجود لینک یا فایل پیوست
* برچسب Spam یا Non-Spam برای هر ایمیل

### 2. داده‌ها را چگونه ذخیره می‌کنید؟

داده‌ها را در یک پایگاه داده یا فایل‌هایی مانند CSV یا JSON ذخیره می‌کنیم تا به راحتی قابل پردازش باشند.

### 3. چه مشکلاتی ممکن است در داده‌ها وجود داشته باشد؟

* داده‌های اشتباه یا ناقص
* تعداد کم نمونه‌های هرزنامه یا ایمیل عادی
* وجود داده‌های تکراری
* برچسب‌گذاری نادرست ایمیل‌ها

### 4. چگونه متوجه می‌شوید مدل عملکرد خوبی دارد؟

مدل را با داده‌های جدید آزمایش می‌کنیم و اگر بیشتر ایمیل‌های هرزنامه را درست تشخیص دهد و ایمیل‌های عادی را به اشتباه Spam تشخیص ندهد، عملکرد آن مناسب است.
